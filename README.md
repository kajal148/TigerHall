# TigerHall Content

To install all the dependencies
```bash
npm install
```

To run the project
```bash
npm start
```

To run all the test cases
```bash
npx install playwright
```
```bash
npm run test-app
```


## Packages and versions

Node version  >= 16

Chakra UI > 2.8

Used Parcel as bundler.

For State Management - Context API

Javascript Library - React
